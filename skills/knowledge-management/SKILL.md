# 知识管理技能

## 功能概述
系统化记录、分类和检索知识，构建个人知识库。

## 核心功能
1. **知识记录**: 灵感、文章、学习笔记
2. **智能分类**: 自动标签和分类
3. **知识检索**: 全文搜索和语义查询
4. **学习跟踪**: 学习进度和效果评估

## 知识类型
- 📝 **灵感记录**: 创意想法和思考
- 📚 **文章收藏**: 重要文章和资料
- 🎓 **学习笔记**: 学习内容和心得
- 💡 **项目想法**: 项目规划和思路

## 使用示例
```
/record 灵感: AI Agent应该有个性化记忆
/search 机器学习
/stats 学习进度
/category 添加标签: AI基础
```

## 技术架构
- 存储: Markdown文件 + 元数据
- 搜索: 全文搜索 + 语义搜索
- 分类: 关键词匹配 + AI分类
- 界面: 命令行 + 飞书集成
