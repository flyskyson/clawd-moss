## 🔧 使用指南

### 1. 内容记录流程

#### 方式A：通过飞书记录
```
您在飞书发送内容 → MOSS智能处理 → 保存到知识库
```

**支持的内容类型**：
- **灵感想法**："记录灵感：AI Agent应该有个性化记忆"
- **文章收藏**："收藏文章：关于Clawdbot扩展的最新研究"
- **学习笔记**："学习笔记：机器学习基础概念总结"
- **项目想法**："项目想法：开发智能知识管理工具"
- **工作记录**："工作记录：今天完成了飞书安全配置"

#### 方式B：直接编辑文件
- 使用Obsidian或其他Markdown编辑器
- 按照模板格式创建新文件
- 保存到相应分类目录

### 2. 内容检索方式

#### 方式A：通过MOSS检索
```
您在飞书询问："我之前关于飞书安全配置的记录"
我：搜索知识库 → 返回相关笔记 → 提供摘要和链接
```

#### 方式B：使用Obsidian搜索
- 利用Obsidian的强大搜索功能
- 通过标签、分类、关键词检索
- 查看知识图谱和关联关系

### 3. 智能分析功能

#### 自动处理：
- **内容摘要**：自动生成文章摘要
- **标签推荐**：基于内容分析推荐标签
- **分类建议**：智能分类到合适目录
- **知识关联**：发现和建立概念关联

#### 个性化分析：
- **学习进度**：跟踪知识掌握程度
- **兴趣分析**：分析学习兴趣和偏好
- **短板识别**：识别知识薄弱环节
- **学习建议**：提供个性化学习路径
